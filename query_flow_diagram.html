API Layer
2

FastAPI Endpoint (app.py)

/api/query → Create/get session → Call rag_system.query()

QueryRequest → QueryResponse

Internal Call
RAG System
3

RAG Orchestrator (rag_system.py)

Build prompt → Get conversation history → Call AI with tools

Prompt: "Answer this question about course materials: {query}"
+ Conversation history + Available tools
← AI response
AI Request
AI Layer
4

Claude API (ai_generator.py)

Send to Claude → If tool_use → Execute tools → Get final response

Anthropic API call with system prompt + tools
Tool execution loop if needed
← Tool results
Tool Execution
Tool Layer
5

Search Tools (search_tools.py)

CourseSearchTool → Execute search → Format results → Track sources

tool_manager.execute_tool("search_course_content")
{ query, course_name?, lesson_number? }
← Search results
Vector Search
Vector Store
6

Vector Operations (vector_store.py)

Semantic search → Apply filters → Return ranked results with metadata

SearchResults: documents + metadata + distances
Course/lesson filtering applied
← Query results
Database Query
Database
7

ChromaDB

Vector similarity search in course_content collection

Embedding-based similarity search
Returns: docs, metadatas, distances
Data Retrieval

Key Data Transformations:

  • User Query → HTTP POST → QueryRequest → RAG prompt
  • AI Decision → Tool use request → Search parameters
  • Vector Search → Ranked documents → Formatted results
  • Tool Results → Claude synthesis → Final response
  • Sources Tracking → UI display → User sees context

Response Flow (Steps 7→1):

ChromaDB returns vectors → Vector Store formats → Search Tool adds context → Tool Manager collects → AI Generator synthesizes → RAG System packages → FastAPI responds → Frontend renders with sources
